# Utilities

## Zip File Checker

For FTP based TMSs, the zip files for translation must conform to the following:
* https://wiki.lingoport.com/Zip_Files_For_Prep_and_Import#Example_of_the_Zip_File_Structure,_One_Zip_per_Locale

Run the lz_zip_checker.sh on the translated zip file before pushing it to the FTP 'from_translation' directory
